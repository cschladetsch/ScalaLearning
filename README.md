# Scala Learning (Pre‑Onboarding)

This is a minimal, focused learning workspace for Scala 3, aimed at an experienced C/C++/C# developer starting a new job next week. The goal is to build reading fluency and reasoning skills quickly, so you can understand and critique AI/vibe‑coding output and the team’s existing codebase.

## What’s inside
- `AGENTS.md`: instructions for contributors/agents.
- `LEARNING_PATH.md`: milestones and weekly focus.
- `EXERCISES.md`: short drills and mini‑projects.
- `RESOURCES.md`: curated references to fill in later.

## How to use
- Follow the milestones in `LEARNING_PATH.md`.
- Do the code‑reading drills first, then hands‑on exercises.
- Keep examples small and self‑contained.
